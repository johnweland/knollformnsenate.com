## Suggested Structure

Use short paragraphs that are easy to scan on the website.

- Keep the `title` concise and clear.
- Keep the `summary` to one sentence.
- Use the `category` to label the post.
- Use an ISO-style `date` like `2026-04-16` so articles sort correctly.
- Set `readTime` to something like `2 min read`.
- Set `image` to a path in `public/` like `/candidate-placeholder.svg` or `/news/townhall.webp` if you want a header image.
